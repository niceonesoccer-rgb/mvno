프로덕션 서버 업로드 파일 목록 (최종)
==========================================

[필수 업로드 파일 - 4개]

1. includes/data/product-functions.php
   - application_id를 정수로 명시적 변환하여 DB 저장
   - 저장 전 로그 추가
   - 경로: includes/data/product-functions.php

2. api/submit-review.php
   - application_id 입력 검증 강화 (null, 빈 문자열, 'null' 문자열 체크)
   - 경로: api/submit-review.php

3. includes/components/mno-sim-order-card.php
   - application_id 검증 추가 (null, 빈 문자열 체크)
   - 경로: includes/components/mno-sim-order-card.php

4. mypage/mno-sim-order.php
   - application_id 검증 및 디버깅 로그 추가
   - 리뷰 작성 시 application_id가 유효한지 확인
   - 경로: mypage/mno-sim-order.php

[선택사항]

5. check-mno-sim-review-logs.php
   - 새로 만든 로그 확인 스크립트
   - 프로덕션에서도 로그 확인 가능
   - 경로: check-mno-sim-review-logs.php

==========================================
업로드 방법:
1. FTP 클라이언트 (FileZilla 등)로 ganadamobile.co.kr 연결
2. 위 4개 파일 (또는 5개)을 프로덕션 서버의 해당 경로에 업로드
3. 기존 파일을 덮어쓰기 (덮어쓰기 전 백업 권장)

업로드 후 확인:
1. 브라우저 캐시 클리어 (Ctrl+F5)
2. 프로덕션에서 리뷰 작성 테스트
3. 브라우저 콘솔(F12)에서 로그 확인
4. DB에서 application_id가 제대로 저장되었는지 확인

주의사항:
- 프로덕션 서버에 db-config-local.php 파일이 있어야 프로덕션 DB에 연결됨
- 업로드 전 기존 파일 백업 권장
