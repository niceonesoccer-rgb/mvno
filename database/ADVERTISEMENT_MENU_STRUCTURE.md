## 💡 광고 시스템 메뉴 구조 제안

### 제안 1: 광고 관리 섹션 추가 (권장) ⭐

기존 구조에 "광고 관리" 섹션을 추가하는 방식입니다.

```
- 대시보드
- 사용자 관리
  - 판매자 관리
  - 회원 관리
- 주문 관리
  - 통신사단독유심
  - 알뜰폰
  - 통신사폰
  - 인터넷
- 상품 관리
  - 통신사단독유심 관리
  - 알뜰폰 관리
  - 통신사폰 관리
  - 인터넷 관리
- 광고 관리 ⬅️ 신규
  - 광고 가격 설정 (카테고리별 관리)
  - 광고 목록 (카테고리별 조회 가능)
  - 입금 관리
    - 입금 신청 목록
    - 무통장 계좌 관리
  - 세금계산서 발행
    - 세금계산서 발행
    - 세금계산서 발행 내역
- 콘텐츠 관리
- 설정
```

**장점:**
- 광고 관련 기능이 한 곳에 모여있어 관리 편리
- 확장성 좋음 (나중에 광고 통계 등 추가 가능)
- 기존 구조와 일관성 유지

---

### 제안 2: 카테고리별 하위 메뉴 (대안)

각 카테고리별로 광고 메뉴를 분산시키는 방식입니다.

```
- 상품 관리
  - 통신사단독유심 관리
    - 통신사단독유심 광고 관리
  - 알뜰폰 관리
    - 알뜰폰 광고 관리
  - 통신사폰 관리
    - 통신사폰 광고 관리
  - 인터넷 관리
    - 인터넷 광고 관리
- 광고 공통 관리
  - 광고 가격 설정
  - 입금 관리
  - 세금계산서 발행
```

**단점:**
- 메뉴가 복잡해짐
- 광고 기능이 여러 곳에 분산됨
- 확장성이 떨어짐

## 📄 페이지 구조

### 1. 광고 가격 설정 (/admin/advertisement/prices.php)

**카테고리별 탭 구성:**
```
[통신사단독유심] [알뜰폰] [통신사폰] [인터넷]
```

**각 탭에서:**
- 시간 단위별, 기간별 가격 설정
- 테이블 형태로 표시
- 가격 수정 및 저장

---

### 2. 광고 목록 (/admin/advertisement/list.php)

**카테고리별 필터/탭:**
```
[전체] [통신사단독유심] [알뜰폰] [통신사폰] [인터넷]
```

**또는 드롭다운 필터:**
```
카테고리: [전체 ▼] [통신사단독유심] [알뜰폰] [통신사폰] [인터넷]
상태: [전체 ▼] [광고중] [광고중지] [광고종료]
```

**표시 정보:**
- 상품명
- 판매자 ID
- 광고 상태
- 로테이션 시간
- 광고 기간
- 광고 금액
- 시작일/종료일

---

### 3. 입금 신청 목록 (/admin/deposit/requests.php)

**필터링:**
```
상태: [전체 ▼] [대기중] [확인됨] [미입금] [거부됨]
```

**표시 정보:**
- 신청일시
- 판매자 ID
- 입금자명
- 입금금액 (부가세 포함)
- 공급가액
- 부가세
- 상태
- 확인일시
- 세금계산서 발행 여부

---

### 4. 무통장 계좌 관리 (/admin/deposit/bank-accounts.php)

**기능:**
- 계좌 등록
- 계좌 수정
- 계좌 삭제
- 활성화/비활성화

---

### 5. 세금계산서 발행 (/admin/tax-invoice/issue.php)

**기능:**
- 기간 설정
- 판매자별 합산 금액 조회
- 세금계산서 발행 처리

---

### 6. 세금계산서 발행 내역 (/admin/tax-invoice/list.php)

**기능:**
- 발행 내역 목록 조회
- 상세보기 (포함된 입금 내역 확인)
- 취소 (필요시)
