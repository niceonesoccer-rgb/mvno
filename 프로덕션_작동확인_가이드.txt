프로덕션 서버 작동 확인 가이드
================================

1. 브라우저 콘솔에서 확인 (F12)
   - 리뷰 버튼 클릭 시:
     * "리뷰 버튼 클릭 - application_id: XXX, product_id: XXX" 로그 확인
   - 리뷰 작성 시:
     * "리뷰 작성 - application_id: XXX, type: string" 로그 확인
     * application_id가 유효한 숫자인지 확인 (null, undefined, '', '0'이면 안됨)

2. 리뷰 작성 테스트
   - http://ganadamobile.co.kr/mypage/mno-sim-order.php 접속
   - 리뷰 작성 버튼 클릭
   - 리뷰 작성 후 제출
   - "리뷰가 작성되었습니다" 메시지 확인

3. DB 확인
   - http://ganadamobile.co.kr/check-mno-sim-review-logs.php 접속
   - 또는 직접 DB에서 확인:
     SELECT id, application_id, product_id, user_id, created_at 
     FROM product_reviews 
     WHERE product_type = 'mno-sim' 
     ORDER BY created_at DESC 
     LIMIT 5;
   - application_id가 0이 아닌 실제 application_id 값으로 저장되었는지 확인

4. 마이페이지에서 확인
   - 리뷰 작성 후 페이지 새로고침 (F5)
   - "리뷰 작성" 버튼이 "리뷰 수정" 버튼으로 변경되었는지 확인

5. 문제 발생 시 확인 사항
   - 브라우저 콘솔에 에러가 있는지 확인
   - application_id가 제대로 전달되는지 확인
   - DB에 리뷰가 저장되었는지 확인
   - application_id가 0으로 저장되지 않는지 확인

================================
확인 결과:
- 작동함: ✅
- 작동 안 함: ❌ (어떤 부분이 문제인지 알려주세요)
