# 더보기 버튼: 10개씩 vs 20개씩 비교 분석

## 시나리오 설정
- **총 게시물**: 5,000개
- **동시 접속자**: 100명
- **사용자 행동 가정**:
  - 10개씩: 평균 3회 클릭 (30개 조회)
  - 20개씩: 평균 2.5회 클릭 (50개 조회) - 초기에 더 많이 보여주므로 더 많은 조회
